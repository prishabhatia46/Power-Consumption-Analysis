# ⚡ Power Consumption Analysis

A machine learning-powered web dashboard built with Flask to analyze and visualize household energy consumption patterns.

**Duration:** May 2025 – July 2025

---

## Overview
This project aims to identify household energy consumption inefficiencies through data analysis and predictive modeling. An interactive web dashboard allows users to explore consumption trends and patterns in real time.

## Features
- Interactive web dashboard built with Flask and Jinja2 templates
- Data preprocessing and feature engineering on household energy data
- Regression models trained to predict and identify consumption inefficiencies
- Dynamic charts and visualizations using Matplotlib
- Real-time data exploration through the browser

## Tech Stack
| Category | Tools |
